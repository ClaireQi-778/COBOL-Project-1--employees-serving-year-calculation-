# CST 8283 Business Programming - Project 1: Employee Data Processor

**Author:** Wenjuan Qi [cite: 127]  
**Course:** CST 8283 Business Programming [cite: 12]  

## Project Overview
This project is a COBOL application designed to manage employee data. [cite_start]The program interactively collects employee information from the user via keyboard input, writes this data to an external sequential text file, and subsequently generates a formatted report displaying only the employees who have served the organization for at least 10.5 years[cite: 48, 55, 127].

## Features
**Interactive Data Entry:** Prompts users to enter employee details field by field, including a prompt to decide whether to continue entering new records[cite: 51, 52].
**Sequential File Processing:** Writes the validated input directly to an external output file (`EMPLOYEES.TXT`)[cite: 48, 129].
**Conditional Reporting:** Re-opens the generated file as an input file, reads the records, and filters the output to only display employees with `YEAR-OF-SVC >= 10.5`[cite: 55, 147].
**Formatted Output:** Generates a structured tabular report with headers for Employee ID, Department, First Name, Last Name, and Years of Service[cite: 132, 133, 134, 135].

## File Structure
`PROJECT1.cbl`: The main COBOL source code file containing the program logic[cite: 127, 128].
`checkDecimalPoint.COB`: A supplementary test program used to experiment with COBOL decimal point formatting (`99V9` and `99.9`)[cite: 1, 2, 3].
`CST 8283 PROJECT 1.pdf`: The official project specification and requirements document.
`PROJECT1.drawio`: The function and hierarchy charts detailing the program's structural logic and routines[cite: 149].
`EMPLOYEES.TXT`: The external data file generated by the program during execution (created in the parent directory `../`)[cite: 129].

## Record Structure
The system processes employee records with the following strict byte structure without spaces[cite: 58, 82]:
**Employee ID:** 6 bytes numeric [cite: 60, 61]
**Department Code:** 3 bytes numeric [cite: 62, 63]
**Last Name:** 20 bytes alphabetic [cite: 64, 65]
**First Name:** 20 bytes alphabetic [cite: 66, 67]
**Years of Service:** 3 bytes numeric with one implied decimal (`99V9`) [cite: 68, 69]
